서비스 이용 가이드

💡

교육디지털원패스(https://edupass.neisplus.kr)에 가입한 계정으로 KERIS의 "AI·디지털 교육자료 포털"에 로그인하면 신청한 자료를 이용할 수 있습니다.

교직원 회원가입

  1. PC에서 교육디지털원패스 접속
  2. [교직원 회원가입] 클릭
  3. 교직원 인증서 로그인 (인증서 선택 → 나이스 아이디 → 인증확인)
  4. 회원 정보 추가 입력 (비밀번호, 이메일 등)
  5. 가입 완료 및 포털 로그인

학생 회원가입

  1. 교육디지털원패스 접속
  2. [학생 회원가입] 클릭 후 약관 동의
  3. 학생 학적 정보 입력 (학교, 학년, 반, 번호, 이름 등)
  4. 본인확인 (휴대폰) 후 추가 정보 입력
  5. 가입 완료 및 포털 로그인
※ 14세 미만 학생은 법정대리인의 동의가 필요합니다.
※ 회원가입 문의: 나이스플러스 고객센터 (1600-7440)
